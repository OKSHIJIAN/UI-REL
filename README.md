# UI-REL Design Review Plugin

Figma 走查插件 - 设计稿与实机截图的差异热力图对比工具。

## 功能特性

### 核心功能
- **Frame 选择**: 在 Figma 中选择任意 Frame，自动捕获设计稿
- **截图上传**: 拖拽或点击上传实机截图 (PNG/JPG/WEBP)
- **差异分析**: 像素级对比，计算 RGB 空间欧氏距离
- **热力图可视化**: 差异越大，颜色越深（蓝→青→绿→黄→红）

### 视图模式
| 模式 | 说明 |
|------|------|
| **热力图** | 在设计稿上叠加差异热力图 |
| **叠加对比** | 半透明叠加显示两张图片 |
| **并排显示** | 左右分栏对比 |
| **仅差异区** | 灰度背景 + 高亮差异区域 |

### 可调参数
- **灵敏度阈值**: 5-80，控制检测精度（值越低检测越细微差异）
- **透明度**: 0-100%，仅对叠加模式生效

### 导出功能
- 一键导出高清热力图报告（2x 分辨率 PNG）

## 安装方式

1. 打开 Figma → Plugins → Development → Import plugin...
2. 选择本目录 (`UI-REL`)
3. 或将整个文件夹复制到 Figma 插件目录

## 使用流程

```
Step 1: 选择设计稿
  └─ 点击 "选择 Figma Frame" → 在画布中选中 Frame

Step 2: 上传实机截图  
  └─ 拖拽/上传截图文件

Step 3: 开始差异分析
  └─ 点击 "开始差异分析" → 查看热力图结果

Step 4: 切换视图模式 & 导出报告
```

## 技术实现

### 差异算法
```
diff = √(ΔR² + ΔG² + ΔB²)

其中:
  ΔR = R_design - R_screenshot
  ΔG = G_design - G_screenshot  
  ΔB = B_design - B_screenshot
```

### 热力图颜色映射
采用 10 阶颜色梯度：
`深蓝 → 蓝 → 青 → 绿青 → 黄绿 → 黄 → 橙 → 红橙 → 红 → 深红`

### 图片预处理
- 自动缩放至相同尺寸进行对比
- 设计稿以 2x 分辨率导出保证清晰度
- 支持 Frame / Component / Instance 类型
